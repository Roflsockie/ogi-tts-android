# Figma Design Prompt: OGI TTS Android App Interface

## Project Overview
Create a modern Android app interface for OGI TTS (Text-to-Speech) application using Material Design 3 principles. The app should have a dark theme and clean, intuitive layout.

## Color Scheme (Based on App's Color.kt)
- **Primary Background:** Dark gray (#121212) - OgiBackgroundDark
- **Surface:** Dark gray-blue (#1E1E1E) - OgiSurfaceDark
- **Primary Color:** Bright blue (#64B5F6) - OgiPrimaryDarkTheme
- **Secondary Color:** Bright teal (#26A69A) - OgiSecondaryDarkTheme
- **Accent:** Orange (#FF6F00) - OgiAccent
- **Text Primary:** White (#FFFFFF)
- **Text Secondary:** Light gray (#B3B3B3)
- **Error:** Red (#D32F2F) - OgiError
- **Success:** Green (#4CAF50)

## Typography
- **Headline Large:** Roboto, 32px, Bold, White
- **Headline Medium:** Roboto, 24px, Medium, White
- **Title Large:** Roboto, 22px, Regular, White
- **Body Large:** Roboto, 16px, Regular, Light Gray
- **Body Medium:** Roboto, 14px, Regular, Light Gray
- **Label Large:** Roboto, 14px, Medium, White

## Interactive States

### Button States
- **Normal:** Default colors
- **Pressed:** 10% darker with ripple effect
- **Disabled:** 50% opacity, gray colors
- **Focused:** Blue border highlight

### Text Field States
- **Normal:** Light gray border
- **Focused:** Blue border (2px)
- **Error:** Red border
- **Disabled:** Gray background, reduced opacity

### Dropdown States
- **Collapsed:** Default surface color
- **Expanded:** Blue border, show menu items
- **Menu Items:** Hover effects, selection highlight

## Animations & Transitions
- **Button Press:** 100ms scale down to 95%
- **Screen Transitions:** 300ms fade in/out
- **Dialog Appearance:** 200ms scale from 90% to 100%
- **Progress Indicators:** Smooth rotation
- **List Items:** 150ms slide in from bottom

## Icons & Assets
- Use Material Design Icons
- Consistent 24x24px icon size in buttons
- Custom OGI TTS logo for welcome screen
- Sound wave visualization elements

## Responsive Design
- **Mobile:** Standard Android layout
- **Tablet:** Center content with max width 600px
- **Landscape:** Adjust button layouts, stack vertically if needed

## Accessibility
- All interactive elements have proper touch targets (48x48dp minimum)
- Sufficient color contrast ratios
- Screen reader friendly labels
- Keyboard navigation support

## Export Requirements
- Export as Figma components for reusability
- Include variants for different states
- Add auto-layout for responsive behavior
- Create a design system with shared styles
